Smart Traffic Control System

The smart traffic control system is an intelligent traffic light fixture meant to ensure smooth traffic flow. This may be especially helpful in the event of an emergency so that rescue vehicles such as ambulances and fire brigades can have a clear path to reach their destination on time. The syatem constitues an Internet of Things module and a server that stores the data and assists in signal exchange.
